What is DOM and what is the purpose of DOM?

DOM or Document Object Model is a way to interact with and manipulate the HTML. It makes the contents of the HTML page into parts of objects which can then be accessed by Javascript or other language to run code on them, to change or interact with the HTML dinamically

How can you select elements from the DOM?

You Can use

1. document.getElementById('elementId')
It selects by the Id of Element. Used to get only one element as only one element can have an unique Id.
2. document.getElementsByClassName('className')
>It Selects by the Class name of Elements. Can be used to capture more than one element and stored in an array like object.
3. document.getElementsByTagName('tagname');
It Selects by the Tag name of Elements. Can be used to capture more than one element and stored in an array like object.
4. document.querySelector("Tag name"/."Class name"/#"Id name");
It Selects by the ("Tag name"/."Class name"/#"Id name") of Elements. Used to capture just one element. For tag name and class name, it will select only the first element with that tag or class
5. document.querySelectorAll('Tag Name'/.'class Name')
It usually Selects by the Class name or tag name of Elements. Can be used to capture more than one element and stored in an array like object. It can be used with other unique selectors also.

What is "Event Bubble" ?

When there is an event in the DOM, Event bubble helps to propagate upwards in the DOM tree.
The event getting triggered on one element will also trigger the parrents of that element.
And thus will it go upwards in the DOM tree.
Event Bubble maintains order also.
First the main element is triggered, then it's parent, then it's parent and so on.
we can manually write code to stop this bubble where we want it to.

Waht is CallBack function used for?

If a function is used as a parameter in another function then it is called a CallBack function.
A function acts as a variable in the callBack parameter.
A function can do different tasks based on which function is passed to it as a parameter.
